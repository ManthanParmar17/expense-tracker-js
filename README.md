# Expense Tracker

A simple Expense Tracker web application built using **HTML, CSS, and JavaScript**.  
This project allows users to track their income and expenses, calculate their balance, and store transaction data using browser local storage.

## Features

- Add income and expense transactions
- Automatic balance calculation
- Transaction history list
- Data stored using localStorage
- Simple and clean user interface

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Enter a description of the transaction.
2. Enter the amount.
3. Use a **positive number for income** and a **negative number for expenses**.
4. Click **Add Transaction**.
5. The balance, income, and expenses will update automatically.

## Example

- Salary → +5000
- Food → -200
